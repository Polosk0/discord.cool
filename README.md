# discord.cool

## Description

Projet Discord bot/script.

## Installation

```bash
# Cloner le repository
git clone https://github.com/Polosk0/discord.cool.git
cd discord.cool

# Installer les dépendances
pnpm install
```

## Configuration

Copiez le fichier `.env.example` vers `.env` et configurez vos variables d'environnement.

## Utilisation

```bash
# Démarrer le projet
pnpm dev
```

## Structure du projet

```
discord.cool/
├── src/           # Code source
├── public/        # Fichiers publics
├── docs/          # Documentation
```
